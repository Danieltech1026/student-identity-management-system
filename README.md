# Student Identity Management System (SIMS)

A PHP and MySQL web application for managing student records and generating student ID cards.

## Features
- Admin Login
- Add/Edit/Delete Students
- Upload Student Photos
- Search Students
- Student ID Card Generation
- PDF Download

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- XAMPP

## How to Run
1. Copy project folder into htdocs
2. Start Apache and MySQL in XAMPP
3. Import sims.sql into phpMyAdmin
4. Visit:
http://127.0.0.1/sims
